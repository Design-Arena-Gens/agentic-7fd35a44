# Film Scenario Architect

Film Scenario Architect is a Next.js web application designed to help screenwriters and directors translate a written scenario into a structured production blueprint. Capture story metadata, craft character arcs, and outline scenes across acts with instant pacing insights.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to start planning.

## Available Scripts

- `npm run dev` – Start the development server.
- `npm run build` – Create a production build.
- `npm start` – Run the production server.
- `npm run lint` – Lint the project with ESLint.

## Project Highlights

- Structured forms for story metadata, character profiles, and scene beats.
- Automatic act breakdowns with runtime and pacing metrics.
- Insight panel that summarizes dramatic flow in Arabic guidance.
- Production notes area for director-level instructions.
